# лабораторная матан 1

в проекте:

- `main.py` — запуск программы
- `tokenizer.py` — токенайзер
- `latex_parser.py` — парсер
- `ast_nodes.py` — дерево выражений и правила дифференцирования

умеет:

- обычная производная
- многократная производная через `--order`
- частная производная через `--var`
- вывод токенов через `--tokens`
- отключение упрощения через `--no-simplify` (для отладки)

быстрый запуск:

```bash
python3 main.py "\sin(x^2+1)"
```

можно запускать еще чрез консоль (как просили)

---
примеры:

## пример 1

обычная производная.

команда:

```bash
python3 main.py "\sin(x^2+1)"
```

результат:

```text
2x\cos(x^{2}+1)
```

## пример 2

частная производная по `y`.

команда:

```bash
python3 main.py "x^2+y^3" --var y
```

результат:

```text
3y^{2}
```

## пример 3

производная второго порядка.

команда:

```bash
python3 main.py "\sin(x^2+1)" --order 2
```

результат:

```text
2\cos(x^{2}+1)-2x\cdot 2x\sin(x^{2}+1)
```

## пример 4

логарифм по основанию 2.

команда:

```bash
python3 main.py "\log_{2}(x)"
```

результат:

```text
\frac{1}{x\ln(2)}
```

## пример 5

дробь и логарифм.

команда:

```bash
python3 main.py "\frac{x^2+1}{\ln(x)}"
```

результат:

```text
\frac{2x\ln(x)-\left(x^{2}+1\right)\frac{1}{x}}{\left(\ln(x)\right)^{2}}
```

## пример 6

экспонента.

команда:

```bash
python3 main.py "e^x"
```

результат:

```text
e^{x}
```

## пример 7

степень, где и основание, и показатель зависят от `x`.

команда:

```bash
python3 main.py "x^x"
```

результат:

```text
x^{x}\left(\ln(x)+1\right)
```

## пример 8

вывод токенов.

команда:

```bash
python3 main.py "\log_{2}(x)" --tokens
```

результат:

```text
токены:
   COMMAND  log
UNDERSCORE  _
    LBRACE  {
    NUMBER  2
    RBRACE  }
    LPAREN  (
     IDENT  x
    RPAREN  )
       EOF

\frac{1}{x\ln(2)}
```

## пример 9

пример ошибки, если не поставить скобки у функции.

команда:

```bash
python3 main.py "\sin x"
```

результат:

```text
ошибка: для \sin нужны круглые скобки, например \sin(x)
```

## пример 10

отключение упрощения (флаг `--no-simplify`).  
команда без флага даёт упрощённый результат:

```bash
python3 main.py '\sin(0) + 2*x*1 + 0*y + (-1)*(x^2+1-1)'
```

результат:

```text
2-2x
```

с флагом `--no-simplify` все промежуточные шаги и нулевые слагаемые остаются видимыми:

```bash
python3 main.py --no-simplify '\sin(0) + 2*x*1 + 0*y + (-1)*(x^2+1-1)'
```

результат:

```text
0\cos(0)+\left(0x+2\cdot 1\right)1+2x\cdot 0+0y+0\cdot 0+\left(-0\right)\left(x^{2}+1-1\right)+\left(-1\right)\left(2x^{1}\cdot 1+0-0\right)
```

## парсер

`latex_parser.py`.

разбирает токены по приоритетам операций:

- сначала суммы и разности
- потом умножение и деление
- потом степени
- потом унарный минус
- потом числа, переменные, группы и функции

поддерживает:

- `\frac{a}{b}`
- `\sin(x)`, `\cos(x)`, `\tan(x)`, `\cot(x)`
- `\sinh(x)`, `\cosh(x)`, `\tanh(x)`, `\coth(x)`
- `\exp(x)`, `\ln(x)`
- `\log_{n}(x)`

поддерживается только запись со скобками: `\sin(x)`.

## дерево выражения

`ast_nodes.py`.

есть отдельные классы для:
- числа
- константы `e`
- переменной
- унарного минуса
- сложения
- вычитания
- умножения
- деления
- степени
- элементарных функций
- натурального логарифма
- логарифма по основанию

---
для выражения вида `f(x)^g(x)` используется общая формула через логарифм:

```text
(f(x)^g(x))' = f(x)^g(x) * (g'(x) ln(f(x)) + g(x) * f'(x) / f(x))
```

## еще есть упрощение результата

после дифференцирования выражение упрощается
реализованы простые полезные упрощения:

- `0 + a = a`
- `a + 0 = a`
- `a - 0 = a`
- `a - a = 0`
- `0 * a = 0`
- `1 * a = a`
- `a * 1 = a`
- `a / 1 = a`
- `a / a = 1`
- `a^1 = a`
- `a^0 = 1`

упрощение можно отключить флагом `--no-simplify`, чтобы увидеть все шаги дифференцирования без преобразований
